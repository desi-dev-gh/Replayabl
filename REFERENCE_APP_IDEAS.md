# Reference App Ideas

## 1. Whiteboard / Diagram Editor

Why it is attractive:

- clear action vocabulary
- easy to visualize branches
- obvious human + AI collaboration
- manageable technical complexity

Typical actions:

- add node
- move node
- connect nodes
- group nodes
- relabel node
- auto-layout
- summarize diagram

## 2. Slide Builder

Why it is attractive:

- human + AI collaboration is very natural
- content generation + layout changes are easy to demonstrate
- replay history is understandable

Typical actions:

- create slide
- add title
- add bullet list
- add image
- change theme
- reorder slides
- rewrite section

## 3. Workflow Builder

Why it is attractive:

- highly structured domain
- useful for enterprise scenarios
- AI can help generate or optimize workflows

Typical actions:

- add step
- add field
- define condition
- connect steps
- add validation rule
- publish workflow

## 4. Form Builder

Why it is attractive:

- constrained and structured
- good fit for event-first systems
- easy to validate

Typical actions:

- add field
- rename field
- change type
- add validation
- reorder fields
- create section

## 5. Image Editor

Why it is interesting:

- strong human + AI collaboration story
- highly visible demo value

Why it is risky early:

- more complex rendering and state handling
- raster semantics are harder
- model actions need tighter validation

Recommendation:

Start with whiteboard, slides, or workflow builder before attempting image editing.
